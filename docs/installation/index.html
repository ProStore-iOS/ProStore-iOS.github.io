Installation

Install ProStore

ProStore Web

The first step to install ProStore, is to visit ProStore Web (https://prostore-ios.github.io) on your iPhone or iPad, and click Download and then Install, and then Install again once the popup appears. This will install ProStore, so you can go to your home screen to check if it’s installed. There may be a few outcomes from this:

  1. ProStore will open and the Setup page will appear.
  2. An Untrusted Enterprise Developer popup will appear.
  3. An Unable to verify popup will appear.

If ProStore opens to the setup screen, continue to the Setup documentation. Otherwise, navigate the the specific outcome below.

Untrusted Enterprise Developer

If you see the Untrusted Enterprise Developer popup, this is good as it means that the certificate is not detected as revoked, and will install as soon as you’ve trusted it.

To trust the certificate, go to Settings → General → VPN & Device Management, and under Enterprise App, tap on the certificate and click Trust "Certificate Name" and then Trust. If another popup appears, you may have to restart your device, so press Allow & Restart, and follow the steps to trust the certificate.

Now the certificate will be trusted, and ProStore will open to the setup screen.

Unable to verify

If you see the Unable to verify popup, this means the certificate that ProStore was installed with was revoked. Make sure you followed the Anti-Revoke DNS steps if you haven’t already, and then continue.

Uninstall ProStore before continuing, otherwise it won’t work to install ProStore with a different certificate as mentioned below.

On ProStore Web, press on Certificate Manager, which will bring you to the certificate manager page. If every certificate says Revoked and none are signed, you will have to just try installing ProStore with them and hope the Anti-Revoke DNS works. If there is a Signed certificate, remember what it’s called.

Go back to ProStore Installer, and tap Advanced Installer. If there was a signed certifiate, select that in the certificate selection box. If not, try each certificate one by one. Click the Download button, and wait for it to download, and then press Install on this device and then Install, to re-install ProStore with the selected certificate. After every change of cert, remember to uninstall ProStore before re-installing.

If it still says Unable to verify, go through and re-install ProStore with each cert to see if any of them work. If none of them work, unfortunately you will not be able to install ProStore with an enterprise cert.
